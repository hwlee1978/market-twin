AI Market Twin

정부 통계 기반 AI 페르소나로 검증하는 글로벌 진출 의사결정 SaaS

제출일: 2026년 5월 13일  ·  작성: Mr.AI Inc. (Market Twin)  ·  분야: AI·디지털 / 기술사업화

1. 제품·서비스의 개요

AI Market Twin은 한 국가의 제품(K-콘텐츠·K-뷰티·K-푸드·D2C 브랜드 등)이 24개 후보 해외 시장에서 성공할 확률을 정부 통계 기반 AI 페르소나 200명 × 다중 LLM 앙상블로 24시간 안에 산출하는 B2B SaaS입니다. 기존 글로벌 컨설팅 펌(맥킨지·베인 등)이 $500K~$1M, 6개월에 제공하던 해외 진출 타당성 검증을, 정부 통계청 데이터에 앵커된 멀티 LLM 시뮬레이션으로 1/200 비용, 1/180 기간에 제공합니다.

한 줄 정의. 정부 통계 기반의 200명 AI 페르소나가 24개국에서 K-product의 진출 가능성을 24시간 안에 검증하고, 의사결정자가 즉시 사용할 수 있는 정량 점수·근거·출처를 출력하는 "디지털 시장조사 SaaS".

2. 구현 방식

2.1 6단계 파이프라인

┌──────────────────┐  ┌──────────────────┐  ┌──────────────────┐
│ ① 페르소나 풀     │→ │ ② 시장 프로파일    │→ │ ③ 시뮬레이션      │
│  (정부 통계 기반)  │  │  (Tavily 그라운딩) │  │  (Multi-LLM)     │
└──────────────────┘  └──────────────────┘  └──────────────────┘
                                                    ↓
┌──────────────────┐  ┌──────────────────┐  ┌──────────────────┐
│ ⑥ 리포트 출력     │← │ ⑤ 검증·환각 필터   │← │ ④ 앙상블 집계     │
│  (PDF/대시보드/   │  │  (출력 단 sanitizer)│ │  (3개 LLM 합의)   │
│   공유 링크)       │  │                  │  │                  │
└──────────────────┘  └──────────────────┘  └──────────────────┘
  1. 페르소나 풀 — KOSIS(한국 통계청)·BLS(미 노동통계국)·e-Stat(일본 총무성) 등 24개국 공식 통계의 직업·소득·연령 분포를 그대로 따르는 200~10,000명 페르소나를 워크스페이스별로 생성·저장.
  2. 시장 프로파일 — Tavily 웹 검색으로 시장 규모·경쟁사·규제 현황을 실시간 수집하여 LLM에 사실로 주입.
  3. 시뮬레이션 — 풀에서 샘플링한 200명 페르소나가 24개 후보 시장 각각에서 제품에 반응. 구매 의향(0~100), 거부 요인, 신뢰 요인을 1인칭으로 출력.
  4. 앙상블 집계 — Claude·GPT·Gemini 3개 LLM이 독립적으로 시뮬레이션하고 합의(consensus)를 추출.
  5. 검증·환각 필터 — 출력 단에서 경쟁사 가격 방향 역전, 시장 규모 출처 범위 이탈 등을 자동 감지·차단.
  6. 리포트 출력 — 웹 대시보드 + 20~40p PDF 리포트 + CSV export + 만료 토큰 공유 링크를 한·영 동시 제공.

3. 특징

3.1 24개국 동시 시뮬레이션

한 번의 시뮬레이션에서 다음 항목을 24개 후보 시장 전체에 대해 동시 산출합니다.

  • finalScore — 0~100 진출 적합도 종합 점수 + 통합 표준편차(within-sim × across-sim, law of total variance)
  • demandScore / competitionScore / CAC — 채널 비용 기반 서버 측 계산값(LLM 추정값 아님)
  • Top objections / trust factors — 페르소나 거부·신뢰 요인을 12개 코드 분류(taxonomy)로 집계
  • 6차원 컴포넌트 — marketSize · culturalFit · channelMatch · priceCompat · competition · regulatory

3.2 페르소나 풀 재사용 레이어

  • 워크스페이스 단위로 200~10,000명 페르소나 사전 생성
  • 신규 시뮬은 풀에서 샘플링 + 반응만 LLM 호출 → 시뮬당 LLM 비용 80% 절감
  • 8개 직업 카테고리, 슬롯 기반 다양성 보장

3.3 다중 출력 형식 (Multi-surface)

출력주 사용처
웹 대시보드의사결정자 1차 검토

CSV export후속 분석·정부기관 자료 첨부

3.4 진출 원산지(Originating Country) 양방향 지원

한국 기업뿐 아니라 24개국 어디서든 다른 23개국으로의 진출을 시뮬레이션 가능합니다. 일본→동남아, 미국→한국 등 양방향 검증을 지원하여 해외 본사·법인을 둔 고객도 동일 도구로 활용 가능합니다.

4. 독창성

4.1 정부 통계 기반 페르소나 합성

기존 AI 시장조사 도구는 LLM 학습 데이터에 잠재된 "평균적 인간"을 가정합니다. AI Market Twin은 각 국가의 공식 통계청 데이터(소득 분포, 직업 분포, 연령 구조)에서 시작합니다. 

KOSIS (한국 통계청)        ┐
BLS   (미 노동통계국)        │
e-Stat (일본 총무성)         ├→ 페르소나 합성 ── 24개국 × 200명
DESTATIS (독일 연방통계)     │   (직업·소득·연령 분포 일치)
INSEE (프랑스 통계청)        │
… 19개국 추가 통계 시드      ┘

결과적으로 "30대 한국 여성"이 아니라 "KOSIS 2024 분포에 따른 30~34세 서울 거주 사무직 월소득 450만원 여성"으로 페르소나가 합성되어, 시뮬레이션 결과의 사회·경제적 대표성이 확보됩니다.

4.2 Multi-LLM 앙상블 + 출력 단 환각 검증

국가기관 제출 시 가장 중요한 신뢰성 보장 메커니즘입니다. 3개 LLM(Claude·GPT·Gemini)이 독립적으로 시뮬을 돌리고 합의를 추출한 뒤, 출력 단 sanitizer가 3중 방어를 수행합니다. 

┌──────────────────────────────────────────────────────────┐
│ Layer 1: 프롬프트 측 사실 블록                            │
│   "COMPETITOR ACTUAL PRICES (FACT — do not guess)"       │
└──────────────────────────────────────────────────────────┘
                          ↓
┌──────────────────────────────────────────────────────────┐
│ Layer 2: 출력 측 환각 필터                                 │
│   · 경쟁사 가격 역전 자동 감지 (방향성 잘못 시 drop)        │
│   · marketSize Tavily 그라운딩 검증 (snippet 범위 밖       │
│     0.5~2× 벗어나면 "⚠ 출처와 차이 큼" 경고 표시)          │
└──────────────────────────────────────────────────────────┘
                          ↓
┌──────────────────────────────────────────────────────────┐
│ Layer 3: 분포 통계 검증                                    │
│   · combinedStd = √(E[within²] + across²) — 진짜 noise   │
│   · 12개 카테고리 taxonomy로 거부 요인 분산 방지           │
└──────────────────────────────────────────────────────────┘

4.3 페르소나 풀 재사용 + Reaction-only Path

대부분의 AI 시장조사 도구는 매번 페르소나를 새로 생성하여 비용이 비례 폭증합니다. AI Market Twin은 페르소나를 한 번 만들어 풀에 저장하고, 새 시뮬은 풀에서 샘플링 + 제품에 대한 반응만 LLM에 호출합니다. 같은 페르소나 풀로 100개 제품을 검증해도 풀 재생성은 0회입니다.

4.4 정부기관 제출 수준의 출력 일관성

모든 표시 surface(웹 대시보드 / PDF / CSV / 공유 링크)에서 동일한 숫자가 표시되도록 일관성을 강제합니다.

  • 채널 비용 기반 CAC가 PDF·웹·CSV·공유 4곳에서 모두 같은 값
  • 페르소나 거부 분류(taxonomy)가 4개 페이지 surface에서 동일 카테고리
  • 표준편차가 within + across 통합값으로 4 surface 통일

"어디서 보느냐에 따라 다른 숫자" 문제를 원천 차단하여, 정부기관 검토자가 PDF 첫 페이지와 CSV 데이터를 교차 검증했을 때 항상 일치하도록 설계되었습니다.
