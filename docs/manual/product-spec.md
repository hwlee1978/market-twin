## 1. 제품 개요

| 항목 | 내용 |
|---|---|
| 제품명 | **Market Twin** (AI 출시 시뮬레이션) |
| 분류 | SaaS (지식서비스 / 클라우드 소프트웨어) |
| 용도 | 한국 제품의 **해외 진출을 출시 전에 검증**하는 AI 시장 시뮬레이션 |
| 제공 형태 | 웹 기반(브라우저), 별도 설치 불필요 |
| 지원 언어 | 한국어 / 영어 |
| 접속 | `app.markettwin.ai` |

**핵심 개념**: 진출 후보국의 현지 소비자를 정부 통계 기반으로 모사한 다수의 AI 페르소나가, 사용자의 제품·가격·크리에이티브에 어떻게 반응할지를 멀티 LLM으로 시뮬레이션하여 **출시 성공 가능성·추천 시장·적정 가격·리스크·실행 전략**을 산출합니다.

## 2. 주요 기능 명세

| # | 기능 | 설명 |
|---|---|---|
| F-01 | 출시 성공 점수 | 수요·CAC·경쟁 강도 기반 0~100점 평가 |
| F-02 | 다국가 시장 비교 | 후보국별 점수·득표율 산출, 추천 진출국 Top-2 도출 |
| F-03 | AI 페르소나 시뮬레이션 | 정부 통계 기반 현지 소비자 페르소나의 구매의향·반응·거부요인 분석 |
| F-04 | 페르소나 대화(Q&A) | 개별 페르소나에게 자연어 질문 → 1인칭 답변 |
| F-05 | 가격 최적화 | ±50% 범위 가격 탄력성 곡선, 매출 최대점·권장가 산출 |
| F-06 | 경쟁사 분석 | 경쟁사 자동 발굴(AI), 위협도·강점/약점·가격 비교 |
| F-07 | 크리에이티브 검증 | 광고 컨셉/시안(이미지)별 0~100점 + 시장별 강·약점 (vision-AI) |
| F-08 | 규제 사전 검증 | 법적 판매 불가 시장 자동 제외 |
| F-09 | 시장 규모(TAM) 분석 | 추천국 시장 규모·성장 추세·도달 세그먼트(출처 포함) |
| F-10 | 투자/ROI 의사결정 | CAC·M:R·마케팅 예산·예상 매출·손익분기(BREAK-EVEN) |
| F-11 | 리스크 분석 | 종합 리스크 수준 + 개별 리스크(심각도·합의율) |
| F-12 | 추천 액션 매트릭스 | 영향×실행난이도 4분면(Quick Wins/Strategic/Marginal/Avoid) |
| F-13 | PDF 리포트 생성 | 임원용·전체 분석·교차검증 3종 자동 생성 |
| F-14 | 결과 공유 | 공유 링크(만료 설정), 런칭 결과 공유 |
| F-15 | 워크스페이스/팀 | 멀티테넌시, 역할 기반(owner/admin/analyst/viewer) |

## 3. 입력 사양 (프로젝트 생성 6단계)

| 단계 | 입력 항목 |
|---|---|
| 1. 제품 | 템플릿(7종) · 프로젝트명 · 제품명 · 카테고리(10종) · 제품설명(≥10자) |
| 2. 가격·목표 | 기본 가격 + 통화(USD/KRW 등) · 출시 목표(브랜드인지/즉시매출/재구매·리텐션/신규국가) |
| 3. 진출 시장 | 출시 모국 · 진출 후보국 1~12개(5개 내외 권장, 24개국+ 지원) |
| 4. 경쟁사 | 경쟁사명(AI URL 발굴) · 경쟁사 URL (선택) |
| 5. 크리에이티브 | 컨셉(텍스트) · 시안 이미지 업로드(≤4MB)/URL (선택) |
| 6. 검토·실행 | 입력 검토 후 시뮬레이션 실행 |

**지원 카테고리(예)**: 뷰티, 패션, 식품·음료, 건강기능식품, 전자기기, 생활용품, 반려, SaaS·소프트웨어, IP·콘텐츠, 기타.

## 4. 시뮬레이션 엔진 사양

### 4-1. 분석 티어

| 티어 | 시뮬 수 × 페르소나 | 총 페르소나 | 비고 |
|---|---|---|---|
| 초기검증 (Hypothesis) | 3 × 200 | 600 | 무료 체험 기본 |
| 검증 (Decision) | 6 × 200 | 1,200 | |
| **검증분석 PLUS (Decision Plus)** | **15 × 200** | **3,000** | 화면 예시 기준 |
| 심층 (Deep) | 25 × 200 | 5,000 | 멀티 LLM |
| 심층 Pro (Deep Pro) | 50 × 200 | 10,000 | 최대 규모 |

### 4-2. AI 엔진 (멀티 LLM 앙상블)
- **Anthropic Claude · OpenAI GPT · Google Gemini · xAI · DeepSeek** 을 라운드로빈으로 병렬 실행 → 단일 모델 편향을 상쇄하고 신뢰도 등급(STRONG/MODERATE/WEAK)을 산출.
- 다단계 파이프라인: 페르소나 생성 → 국가 점수 → 반응 분석 → 가격 민감도 → 종합(합의 narrative) → 품질 감사.

### 4-3. 데이터 근거 (Grounding)
- 정부·국제 통계: **KOSIS, BLS, e-Stat, UN Comtrade, World Bank, 관세청, KOTRA** 등
- 실시간 시장 컨텍스트: 트렌드·마진 벤치마크(Tavily)
- 규제·경쟁 참조 데이터: 카테고리별 규제·경쟁사 DB
