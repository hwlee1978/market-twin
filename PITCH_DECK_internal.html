품질 Deep-dive · 내부 한정

품질 방어 시스템 — 3-layer voice sanitizer

LLM 페르소나 보이스에서 발견된 3가지 슬립 패턴을 프롬프트 + 런타임 sanitizer 양면으로 차단

Slip Pattern 1 — JP 보이스 누출

  • 일본 페르소나에서 영문 응답 / 한국식 표현 혼입
  • Defense: locale-filter + 프롬프트 ko/ja anchor
  • 런타임 감지 → 재생성 트리거

Slip Pattern 2 — Brand-name bias

  • 페르소나가 자발적으로 브랜드명 (Olive Young, COSRX 등) 호명
  • Defense: brand-name bias 차단 sanitizer
  • 28-브랜드 사전과 cross-check — 자연 멘션만 통과

Slip Pattern 3 — Auto-translation 감지

  • 1인칭 페르소나가 기계번역 톤으로 떨어지는 패턴
  • Defense: auto-translation 감지 휴리스틱
  • 특정 표현 패턴 (직역체) 감지 시 페르소나 재합성

다양성 강제 (sampling-time)

  • 슬롯 기반 직업 분배 — LLM 자유 생성 차단
  • maxTokens 정합 + 시뮬 synthesis date anchor
  • profession diversity 검증 스크립트 상시 가동
  • compare:voice / validate:categories 스크립트

Narrative rewriter: 시뮬-스케일 (200명) → 앙상블-스케일 (10K명/50시뮬) 두 결과를 동일한 임원용 톤으로 통일 — 티어 사이 보고서 품질 격차 제거

티어드 리포트 시스템 — 5 단계

hypothesis 200 1 — 요약만 기본 가설 검증 (싸게)
decision 500 3 — 국가만 표준 국가 우선순위

deep_pro 10,000 50 3 모델 round-robin 전체 + 신뢰구간 풀 + 부록 대형 출시 (전사)
